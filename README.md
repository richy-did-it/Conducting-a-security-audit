# Security Audit for a Small Business

This project demonstrates how to conduct a basic cybersecurity security audit for a small business environment.
The audit is based on concepts taught in the Google Cybersecurity Professional Certificate.

The goal of this project is to identify assets, assess risks and vulnerabilities, and recommend security controls
to improve the organizationâ€™s overall security posture.

## Objectives
- Identify and classify key organizational assets
- Assess potential security risks and vulnerabilities
- Evaluate existing security controls
- Recommend mitigation strategies and security best practices

## Skills Demonstrated
- Security risk assessment
- Asset inventory and classification
- Threat and vulnerability analysis
- Security controls and mitigation planning
- Professional security documentation

## Tools & Concepts
- CIA Triad (Confidentiality, Integrity, Availability)
- Risk assessment principles
- Security auditing fundamentals
- Documentation and reporting

## Scenario
This audit is based on a simulated small business environment and reflects common real-world
security challenges such as limited resources, access control issues, and data protection risks.

## Disclaimer
This project is for educational purposes only and does not represent a real organization.
